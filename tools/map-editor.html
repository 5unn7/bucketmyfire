How to use

Drag a marker or lake to move it — its real lat/lon updates live.

Resize a lake: select it, then drag the ◇ handle on its rim, scroll the wheel over it, or edit area in the inspector.

Rotate / stretch: edit elong & bearing in the inspector.

Freeform a lake: select it → Convert to editable outline. Drag the ● points to shape the shore, double-click a ● to delete, double-click an edge to add one. Revert to ellipse undoes it.

Add a river: ＋ River, then click to drop points along its course; Enter / double-click finishes. Drag a ● to adjust, double-click a ● to delete, set width in the inspector.

Connect a highway: ＋ Highway, then click bases/towns in order; Enter finishes. Select an existing highway to re-route it (✕ a stop, or Connect anchors to extend).

Paint buildings: pick 🏠 Bldgs, hold-drag over the map. Adjust density & kind in the brush panel.

Paint road: pick ── Road, hold-drag a freehand stroke; release to finish.

Erase: pick ⌫ Erase, drag over any building or road to remove it.

Pan: drag empty space, or hold Space+drag anywhere, or middle-mouse drag. Zoom: mouse wheel. Del removes the selected item. Esc deselects / finishes.

Projection, world scale (2100u) and the lake area→radius band match the game exactly. Rivers aren't in regions.ts yet — Export emits a paste-ready block + the interface to wire them in.
